# AI-Based-Supply-Chain-Demand-Forecasting-Performance-Dashboard
Project Overview  This project focuses on analyzing supply chain data and building predictive analytics solutions using Python, Machine Learning, Power BI, and Streamlit.  The objective is to generate business insights, forecast product demand, and improve decision-making through interactive dashboards and predictive models.

Features
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Revenue & Shipping Cost Analysis
Machine Learning Prediction
Demand Forecasting
KPI Dashboard
Streamlit Web App
Power BI Dashboard

Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Streamlit
Microsoft Power BI

Machine Learning Model
Random Forest Regressor
Revenue Prediction
Feature Importance Analysis

Business Insights
Identified top-performing products
Analyzed shipping efficiency
Forecasted future product demand
Evaluated supplier performance
Improved business decision-making through analytics
